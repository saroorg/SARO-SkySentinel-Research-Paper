SARO SkySentinel

A Low-Cost IoT-Based Atmospheric Monitoring and Early Warning System for Smart Cities.

Authors

- Abhishek C A
- Space Application and Research Organisation (SARO)

Abstract

SkySentinel is an IoT-based atmospheric monitoring system designed to collect environmental data and provide early warnings for adverse conditions. The project uses low-cost sensors and cloud technologies to support smart city applications.

Features

- Real-time environmental monitoring
- Temperature and humidity sensing
- Air quality monitoring
- Early warning notifications
- Low-cost and scalable design
